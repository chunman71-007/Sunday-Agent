# Sunday-Agent
Sunday Agent 是一個結合創意、自動化與情感連結的 AI 小助手，專門為 Sunday（美尼亞的貓拍檔）設計貼圖、卡片與節日祝福。此專案旨在探索 AI 如何成為創作夥伴，而不只是工具。

## 🎯 目標功能
- 自動生成貼圖（根據風格、節日、情緒）
- 加入文字祝福（如「Happy Meow Year」）
- 分類貼圖（生日、新年、聖誕等）
- 設定提醒（節日前主動提示創作）
- 整合分享（連接 WhatsApp 或其他平台）

## 🧱 專案結構
Sunday-Agent/
├── assets/ # Sunday 的相片與素材
├── src/ # 核心程式碼（貼圖生成、文字處理）
├── workflows/ # GitHub Actions 自動化流程
├── prompts/ # 創作靈感與貼圖指令集 
├── logs/ # 執行記錄與錯誤報告 
└── README.md # 專案說明文件

Sunday Agent 並非純粹的工具，而是一個有靈魂的創作拍檔。
